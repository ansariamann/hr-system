# Security Policy

## Supported Versions

We currently support the latest version of the repository for security updates.

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |

## Reporting a Vulnerability

Please report any suspected security vulnerabilities privately to the maintainers. Do not open public issues for security vulnerabilities. We will acknowledge receipt of your vulnerability report as soon as possible and strive to provide regular updates until the issue is resolved.
